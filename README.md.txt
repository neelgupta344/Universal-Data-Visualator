# 📊 Universal Data Visualizer

A professional and interactive data visualization dashboard built using **Streamlit**, **Pandas**, and **Plotly**. This application allows users to upload any CSV dataset and automatically generate insightful visualizations without writing any code.

---

## 🚀 Features

* Upload any CSV file
* Load datasets from URLs
* Automatic data type detection
* Interactive visualizations
* Dataset preview and statistical summary
* Correlation heatmap generation
* Responsive dashboard layout

---

## 📈 Supported Visualizations

* Line Chart
* Bar Chart
* Scatter Plot
* Histogram
* Box Plot
* Pie Chart
* Correlation Heatmap

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Plotly Express

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Universal-Data-Visualizer.git
```

Navigate to the project folder:

```bash
cd Universal-Data-Visualizer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

---

## 📊 How to Use

1. Launch the application.
2. Upload a CSV file or provide a CSV URL.
3. Explore the dataset overview.
4. Select a visualization type.
5. Customize axes and filters.
6. Analyze the generated charts and statistics.
